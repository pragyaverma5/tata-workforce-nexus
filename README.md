# Tata Workforce Nexus

A modern Workforce Management Dashboard inspired by Tata Motors, designed to streamline employee management, attendance tracking, leave management, and reporting.

## 🚀 Features
- Secure Login System
- Dashboard Analytics
- Employee Management
- Attendance Tracking
- Leave Management
- Feedback System
- Reports & Analytics

## 🛠️ Tech Stack
- React
- Vite
- JavaScript
- CSS
- React Router
